Little Lemon API Endpoints

Bookings:

GET /api/bookings/?date=YYYY-MM-DD

POST /api/bookings/

Booking detail:

GET /api/bookings/<id>/

PUT /api/bookings/<id>/

DELETE /api/bookings/<id>/

Menu:

GET /api/menu/

POST /api/menu/

GET /api/menu/<id>/

PUT /api/menu/<id>/

DELETE /api/menu/<id>/


Authentication:

POST /auth/users/   # Register

POST /auth/token/login/   # Login & get token
